IFD CommuniTeach

Click for Instructions This app can be used to select dances for the planned "community teach" event. Using the search box, you can select dances you'd like to learn and dances you'd be willing and able to teach. You can see them listed, and remove them, in the sections below. Beneath those lists, you can see a list of all dances proposed by others, including your own. Browsing those can remind you of some other dances that you'd like to learn or teach. You can add them from there, or remove the ones already in your lists.
